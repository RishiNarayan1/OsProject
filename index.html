How to Use

  1. Select a scheduling algorithm from the dropdown menu
  2. Enter the number of processes (1-10) or choose an example set
  3. Fill in the arrival time and burst time for each process
  4. For Priority Scheduling, enter priority values (lower number = higher priority)
  5. For Round Robin, specify the time quantum
  6. Click "Calculate Schedule" to see the results
